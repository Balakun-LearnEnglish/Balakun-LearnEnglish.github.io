Discord



Discord is a social media chat application.
You can use it on mobile (Andorid and iPhone), on a web browser, or as a desktop application on a PC.

With Discord you can speak with friends by exchanging messages or using a microphone.

Discord is a free application and you can download it here: https://discord.com/download
P.S. the web version does not need to be downloaded

How to use it?

Create an account

Discrod is organizaed in "Servers". A "server" is a group of people with a series of channels where to chat and exchange images and other files.

After you downloaded and install it, you should create your account. Crate an account, give it a name (this name will be how people will see you) and add an image to represent yourself (avatar.)

After this you can join a server. Usually you need an invite link to join a server. For example this link is to join the Balakun server: https://discord.gg/Hm2aC9BxRT

Threads

In Discord, you can have a conversation over a message that was posted.
This feature is called Threads, you will continue to chat about the specific message without adding new messages to the main channel.
